# 组合优化理论

本课程为信息与软件工程学院所开设的研究生专业基础课，主要讲授的内容包括：
 - 线性规划：单纯形法、对偶理论、整数规划
 - 网络流：最大流、最小生成树、最短路径
 - 最优指派问题
 - 装箱问题
 - 背包问题
 - 作业排序问题
 - 旅行售货商问题
 - 启发式算法：遗传算法、 退火算法
